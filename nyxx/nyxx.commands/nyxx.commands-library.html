nyxx.commands library

Nyxx DISCORD API wrapper for Dart

Commands sublibrary provides tool for creating commands bots. It also provides more advanced tools for creating polls. and paginated messages Pagination.

Classes

Command
Defines command. Can be placed above method. It defines command handler for specified command. [...]
CommandContext
All command have to inhertit from this class. This class provides variuos helper methods to access discord world more easly Can be overrriden to create groups of commands (modules) or injected into top-level method commands. [...]
CommandExecutionFailEvent
Emitted when command execution fails
CommandParsingFail
Emitted when parsing error occurs
CommandsFramework
Main point of commands in nyx. It gets all sent messages and matches to registered command and invokes its action.
CooldownCache
Handles caching of cooldown
Help
Help annotation allows to provide help command for commands. Help string will be auto generated by framework.
Module
Can be places above class declaration and it means that class is group of commands and all commands will be have prefix with given name (so it just creates module). [...]
Pagination
Handles pagination interactivity. Allows to create pagnated messages from List Factory constructors allows to create message from String directly. [...]
Postprocessor
Preprocessor allows to run code block after successfull command execuction. It gives possibillity to eg. send additional data or do stuff with databases
Preprocessor
Preprocessor allows to run code block just after finding rigt command handler and before even checking for permission. It gives possibillity to check if command can be run (or not) before even starting all the process.
PreprocessorErrorEvent
Emitted when dispatch function fails when checking preprocessors
Remainder
Captures all remaining command text into List<String> or String
Restrict
Defines additional properties which will restrict user access to command All fields are optional and wont affect command execution of not set.
Scheduler
Runs specific code on every target periodically. [...]
Service
Service is base class for services Overriding this class allows to auto-register your services.
TypeConverter<T>
Interface for creating TypeConverter. Overriding this class alows to later register it into CommandsFramework
