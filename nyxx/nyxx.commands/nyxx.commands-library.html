nyxx.commands library

Nyxx - Discord API wrapper for Dart Commands sublibrary

nyxx.commands provides useful tools for fast bot creation. Allows to catch-up messages and create actions to respond to them. Provides tools to adapt other apis and 3-party services to your bot.

Classes

Command
Defines command. Can be placed above method. It defines command handler for specified command. [...]

All command have to inherit from this class. This class provides various helper methods to access discord world more easly Can be overridden to create groups of commands (modules) or injected into top-level method commands. [...]
CommandExecutionError
CommandsFramework
Main point of commands in nyx. It gets all sent messages and matches to registered command and invokes its action.
CooldownCache
Handles caching of cooldown
Module
Can be places above class declaration and it means that class is group of commands and all commands will be have prefix with given name (so it just creates module). [...]
Postprocessor
Preprocessor allows to run code block after successful command execution. It gives possibility to eg. send additional data or do stuff with databases
Preprocessor
Preprocessor allows to run code block just after finding right command handler and before even checking for permission. It gives possibility to check if command can be run (or not) before even starting all the process.
PreprocessorResult
Remainder
Captures all remaining command text into List<String> or String
Restrict
Defines additional properties which will restrict user access to command All fields are optional and wont affect command execution of not set.
Service
Service is base class for services Overriding this class allows to auto-register your services.
TypeConverter<T>
Interface for creating TypeConverter. Overriding this class allows to later register it into CommandsFramework
