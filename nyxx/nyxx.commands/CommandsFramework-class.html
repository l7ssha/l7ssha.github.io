CommandsFramework class

Main point of commands in nyx. It gets all sent messages and matches to registered command and invokes its action.

Constructors

CommandsFramework(String prefix, Client _client)
Creates commands framework handler. Requires prefix to handle commands.

Register commands in current Isolate's libraries. Basically loads all classes as commnads with CommandContext superclass. Performs dependency injection when instantiate commands. And throws Exception when there are missing services
registerLibraryServices() → void
Register all services in current isolate. It captures all classes which inherits from Service class and performs dependency injection if possible.
registerServices(List<Object> services) → void
Register services to injected into commands modules. Has to be executed before registering commands. There cannot be more than 1 dependency with single type. Only first will be injected.
