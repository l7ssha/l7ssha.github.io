nyxx.voice library

Nyxx DISCORD API wrapper for Dart Voice sublibrary wrapps all tool and logic needed to interact with voice. Allows to connect to channel and play music using Lavalink

Classes

Entity
Represents basic lavalink object: Track or Playlist
Player
Manages voice connection for guild. There can be only one player for Guild
PlayerUpdateEvent
Playlist
Playlist contains list of track and basic info about itself.
Stats
Stats object which is emitted every one minute
Track
Represents Lavalink playable track
TrackEndEvent
TrackError
TrackExceptionEvent
TrackResponse
Response of Lavalink api
TrackStuckEvent
VoiceService
VoiceService managers all voice connections. There can be only one instance class.

Functions

destroyPlayer(Player player) → Future<Null>
Destroys player and removes all connections
getPlayer(Guild guild) → Future<Player>
Gets Player instance for guild.
getVoiceService() → VoiceService
Returns instance of VoiceService
init(String clientId, Client client, String yamlConfigFile) → VoiceService
Inits voice service. yamlConfigFile is absolute path to lavalink config file. Returns instance of VoiceService
