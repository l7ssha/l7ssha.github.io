MessageChannel class

Provides abstraction of messages for TextChannel, DMChannel and GroupDMChannel. Implements iterator which allows to use message object in for loops to access messages sequentially.

var chan = client.channels.firstWhere((ch) => ch is TextChannel);

for (var message in chan) {
  print(message.author.id);
}
