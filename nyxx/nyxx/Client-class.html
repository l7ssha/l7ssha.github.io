Client class

Class representing client - it's place to start with. From there you can subscribe to varius Streams to listen to Events Rememeber to close Client before terminating program:

 bot.destroy();

It closes bot connections to discord servers and makes sure that everything is terminated correctly. Client is also used when creating instance of Commands - Commands Framework.
