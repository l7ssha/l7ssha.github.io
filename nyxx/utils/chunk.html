Stream<List<T>> chunk <T>(List<T> list, int chunkSize)

Implementation 

Stream<List<T>> chunk<T>(List<T> list, int chunkSize) async* {
  int len = list.length;
  for (var i = 0; i < len; i += chunkSize) {
    int size = i + chunkSize;
    yield list.sublist(i, size > len ? len : size);
  }
}
