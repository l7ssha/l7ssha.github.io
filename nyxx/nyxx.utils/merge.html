merge<T> function

Stream<T> merge <T>(List<Stream<T>> streams)

Merges list of stream into one stream

Implementation 

Stream<T> merge<T>(List<Stream<T>> streams) {
  int _open = streams.length;
  var c = StreamController<T>();
  for (var stream in streams) {
    stream.listen(c.add)
      ..onError(c.addError)
      ..onDone(() {
        if (--_open == 0) {
          c.close();
        }
      });
  }
  return c.stream;
}
